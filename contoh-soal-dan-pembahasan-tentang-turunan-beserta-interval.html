9. contoh soal turunan dan pembahasannya


Jawab:

Tentukan turunan dari invers fungsi trigonometri [tex]\displaystyle y=2\tan^{-1}\left ( \sqrt{x^2+6x+7}-x-2 \right )[/tex]

Penjelasan dengan langkah-langkah:

Berdasarkan rumus

[tex]\displaystyle y=\tan^{-1}f(x)\rightarrow y'=\frac{f'(x)}{[f(x)]^2+1}[/tex]

dan

[tex]\displaystyle y=\sqrt{f(x)}\rightarrow y'=\frac{f'(x)}{2\sqrt{f(x)}}[/tex]

maka turunan dari [tex]\displaystyle y=2\tan^{-1}\left ( \sqrt{x^2+6x+7}-x-2 \right )[/tex] adalah

[tex]\displaystyle y'=2\cdot\frac{\frac{x+3}{\sqrt{x^2+6x+7}}-1}{\left ( \sqrt{x^2+6x+7}-x-2 \right )^2+1}[/tex]

Karena soal terlalu rumit maka dimisalkan [tex]\displaystyle u=\sqrt{x^2+6x+7}-x[/tex] lalu tentukan x dengan cara sebagai berikut

[tex]\begin{aligned}\sqrt{x^2+6x+7}&\:=u+x\\x^2+6x+7\:&=x^2+2ux+x^2\\(6-2u)x\:&=u^2-7\\x\:&=\frac{u^2-7}{6-2u}\end{aligned}[/tex]

Kembali ke [tex]\displaystyle y'=2\cdot\frac{\frac{x+3}{\sqrt{x^2+6x+7}}-1}{\left ( \sqrt{x^2+6x+7}-x-2 \right )^2+1}[/tex]

Untuk pembilang

[tex]\begin{aligned}\frac{x+3}{\sqrt{x^2+6x+7}}-1&\:=\frac{x+3}{u+x}-1\\\:&=\frac{\frac{u^2-7}{6-2u}+3}{u+\frac{u^2-7}{6-2u}}-1\\\:&=\frac{u^2-7+3(6-2u)}{u(6-2u)+u^2-7}-1\\\:&=-\frac{u^2-6u+11}{u^2-6u+7}-1\\\:&=\frac{-(u^2-6u+11)-(u^2-6u+7)}{u^2-6u+7}\\\:&=-2\cdot\frac{u^2-6u+9}{u^2-6u+7}\end{aligned}[/tex]

maka

[tex]\begin{aligned}y'&\:=2\cdot\frac{\frac{x+3}{\sqrt{x^2+6x+7}}-1}{\left ( \sqrt{x^2+6x+7}-x-2 \right )^2+1}\\y'\:&=-4\cdot\frac{\frac{u^2-6u+9}{u^2-6u+7}}{(u-2)^2+1}\\y'\:&=-4\cdot\frac{u^2-6u+9}{(u^2-6u+7)(u^2-4u+5)}\\y'\:&=-\frac{4u^2-24u+36}{-[u(6-2u)+u^2-7]~[u^2-7+2(6-2u)]}\\y'\:&=-\frac{(6-2u)^2}{-[u(6-2u)+u^2-7]~[u^2-7+2(6-2u)]}\end{aligned}[/tex]

[tex]\begin{aligned}y'&\:=\frac{1}{\frac{u^2-7+2(6-2u)}{6-2u}\cdot\frac{u(6-2u)+u^2-7}{6-2u}}\\y'\:&=\frac{1}{\left ( \frac{u^2-7}{6-2u}+2 \right )\left ( u+\frac{u^2-7}{6-2u} \right )}\\y'\:&=\frac{1}{(x+2)(u+x)}\\y'\:&=\frac{1}{(x+2)\sqrt{x^2+6x+7}}\end{aligned}[/tex]

12. buaatlah contoh soal tentang kalorimeter beserta pembahasannya !


Soal tentang kalorimeter beserta pembahasannya

Pembahasan

Soal Pertama

ke dalam kalorimeter dimasukkan 100 ml air dan 2,14 gram NH4Cl, ternyata suhu larutan turun dari 27 derajat Celcius menjadi 23,5 derajat Celcius. jika kalor jenis air 4,18 J/g derajat Celcius, deltaH pelarutan adalah ΔH = 36,575 kJ/mol

Jawaban

Diket :

V = 100 mL

m NH₄Cl = 2,14 gram

T₁ = 23,5°C

T₂ = 27°C

c = 4,18 J/gram.°C

Ditanya :

ΔH

Jawab :

a. menentukan perubahan suhu

ΔT = 27 - 23,5 = 3,5°C

b. menentukan massa air dalam larutan

рair = 1 gram/mL

jika V = 100 mL

рair  = m/v

1 = m/100

massa air = 100 gram

c. menentukan perubahan kalor (q)

q larutan = m x c x ΔT

q larutan = 100 x 4,18 x 3,5

q larutan = 1.463 J

q larutan = 1,463 kJ

d. menentukan mol NH₄Cl

Mr NH₄Cl = 1. Ar N + 4. Ar H + 1. Ar Cl

Mr NH₄Cl = 1. 14 + 4. 1 + 1. 35,5

Mr NH₄Cl = 53,5

n NH₄Cl = m/Mr

n NH₄Cl = 2,14/53,5

n NH₄Cl = 0,04 mol

e. menentukan perubahan entalpi (ΔH)

Ingat : Perubahan entalpi standar (ΔH) adalah kJ/mol.

sehingga besarnya perubahan entalpi (ΔH) adalah untuk 1 mol larutan  NH₄Cl.

Dari perhitungan diatas dapat disimpulkan untuk 0,0 4 mol NH₄Cl diperlukan kalor sebesar 1,463 kJ.

maka untuk 1 mol NH₄Cl  :

ΔH = [tex]\frac{1}{0,04}[/tex] x 1,463 kJ

ΔH = [tex]\frac{1,463}{0,04}[/tex]

ΔH = 36,575 kJ/mol

Soal Kedua

kalor yang dibebaskan pada pembakaran 1 gram gas C2H2 dapat menaikkan suhu 1 liter air dari 25°C menjadi 37°C.kalor jenis air diket=4,2 J/g°C,maka entalpi pembakaran gas C2H2(dalam kJ/mol) adalah -1310,4 kJ/mol.

Jawaban

Diket :

massa C₂H₂ = 1 gram

massa air = 1 Liter = 1000 mL

T₁ = 25°C

T₂ = 37°C

c = 4,2 J/g°C

Ditanya :

ΔH (kJ/mol)

Jawab :

Menentukan ΔT

ΔT = T₂ - T₁

ΔT = 37 - 25

[tex]\boxed{\boxed{\bold{\Delta T = 12^oC}}}[/tex]

Menentukan q larutan

q larutan = m x c x ΔT

q larutan = 1000 x 4,2 x 12

q larutan = 50.400 J

[tex]\boxed{\boxed{\bold{q larutan = 50,4 kJ}}}[/tex]

Menentukan q reaksi

q reaksi = - q larutan

[tex]\boxed{\boxed{\bold{q reaksi = -50,4 kJ}}}[/tex]

Menentukan ΔH

ΔH = q reaksi

[tex]\boxed{\boxed{\bold{\Delta H = -50,4 kJ}}}[/tex]

Menentukan Mr C₂H₂

Mr C₂H₂ = 2. Ar C + 2. Ar H

Mr C₂H₂ = 2. 12 + 2. 1

Mr C₂H₂ = 24 + 2

[tex]\boxed{\boxed{\bold{Mr~C_2H_2 = 26}}}[/tex]

Menentukan mol C₂H₂

n C₂H₂ = [tex]\frac{massa}{Mr}[/tex]

[tex]\boxed{\boxed{\bold{n~C_2H_2 = \frac{1}{26}~mol }}}[/tex]

Menentukan ΔH (kJ/mol)

ΔH = -50,4 kJ untuk pembakaran gas C₂H₂ sebanyak 1/26 mol, maka entalpi untuk pembakaran gas  C₂H₂ sebanyak 1 mol adalah

ΔH = [tex]\frac{1}{\frac{1}{26} }[/tex] x  -50,4 kJ

ΔH = 26 x -50,4 kJ

[tex]\boxed{\boxed{\bold{\Delta H = - 1310,4kJ}}}[/tex]

23. Tentukan interval naik dan inderval turun dari fungsi F(×)=×³-6×²+9×+1​


Interval naik: x < 1 atau x > 3

Interval turun: 1 < x < 3

Pembahasan

f (x) = x³ - 6x² + 9x + 1

f'(x) = 3x³⁻¹ - 2 . 6x²⁻¹ + 9x¹⁻¹

f'(x) = 3x² - 12x + 9 ..... (÷ 3)

f'(x) = x² - 4x + 3

x² - 4x + 3

(x - 1)(x - 3)

x = 1 ||x = 3

Fungsi naik, jika f'(x) > 0

Fungsi turun, jika f'(x) < 0

Naik pada interval x < 1 atau x > 3

Turun pada interval 1 < x < 3

38. contoh soal glbb/ glb beserta pembahasannya^^


GLB:

Seorang pengendara motor berjalan dengan kelajuan tetap 10 m/s, dalam waktu 1 jam ia telah menempuh jarak sejauh ?

Dik: v = 10 m/s
       t = 1 jam = 3600 s

Dit: s = ?

Jawab:
s = v.t
s = 10 m/s . 3600 s
s = 36000 m
s = 36 km

GLBB:

Seorang pengemudi mobil bergerak dengan percepatan konstan 5 m/s², Jika waktu yang dibutuhkan untuk menempuh jarak s adalah 20 sekon dan kecepatan awal mobil itu sama dengan 0 m/s. Maka nilai s adalah ?

Dik : a = 5 m/s²
        t = 20 s
        Vo = 0 m/s

Dit : s = ?

Jawab:

Rumus GLBB ke II:

s = Vo.t + 1/2 a.t²
s = 0 m/s . 20 s + 1/2 . 5 m/s² . (20 s)²
s = 0 m + 1/2 . 5 m/s² . 400 s²
s = 600 m
s = 0,6 km
