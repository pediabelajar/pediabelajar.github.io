deklarasi

read(lebar);

luas <-- panjang * lebar;

write(luas);

14. contoh implementasi algoritma dan flowchart untuk menghitung luas bangun datar!​


Jawaban:

Berikut adalah contoh algoritma dan flowchart untuk menghitung luas beberapa bentuk bangun datar:

1) Luas Persegi

Algoritma:

1. Masukkan panjang sisi persegi

2. Hitung luas persegi dengan cara sisi persegi dikalikan dengan sisi persegi

3. Tampilkan hasil luas persegi

Flowchart:

![luas persegi](https://i.imgur.com/LWtdtdT.png)

2) Luas Lingkaran

Algoritma:

1. Masukkan nilai jari-jari lingkaran

2. Hitung luas lingkaran dengan cara jari-jari lingkaran dikalikan dengan jari-jari lingkaran, kemudian dikalikan dengan 22/7 (pi)

3. Tampilkan hasil luas lingkaran

Flowchart:

![luas lingkaran](https://i.imgur.com/A4KXVuS.png)

3) Luas Segitiga

Algoritma:

1. Masukkan nilai alas segitiga

2. Masukkan nilai tinggi segitiga

3. Hitung luas segitiga dengan cara alas segitiga dikalikan dengan tinggi segitiga, kemudian dibagi dua

4. Tampilkan hasil luas segitiga

Flowchart:

![luas segitiga](https://i.imgur.com/YXRB6QV.png)

4) Luas Trapesium

Algoritma:

1. Masukkan nilai panjang sisi sejajar atas trapesium

2. Masukkan nilai panjang sisi sejajar bawah trapesium

3. Masukkan nilai tinggi trapesium

4. Hitung luas trapesium dengan cara jumlahkan panjang sisi sejajar atas dan bawah, kemudian dikalikan dengan tinggi trapesium, kemudian dibagi dua

5. Tampilkan hasil luas trapesium

Flowchart:

![luas trapesium](https://i.imgur.com/WLyvGiG.png)

18. flowchart menghitung volume tabung


Flowchart dalam menghitung volume tabung, gambarnya dapat dilihat di lampiran. Algoritmanya yaitu:

Mulai.Tetapkan nilai π (phi) sama dengan 3,14.Masukkan jari-jari tabung.Masukkan tinggi tabung.Rumus volume tabung = π * jari-jari * jari-jari * tinggi.Tampilkan nilai volumenya.Selesai.

Pembahasan

Flowchart adalah bentuk algoritma yang menggunakan beberapa simbol dalam bentuk bangun datar dan garis tanda panah. Simbol-simbol pada flowchart terdiri dari:

Simbol terminal.Simbol input/output.Simbol proses.Simbol keputusan.Simbol arus.

Dalam flowchart yang dibuat, terdiri dari:

Simbol terminal (berbentuk bangun sisi lengkung yang memanjang seperti perti panjang) diisi dengan kata “Mulai”.Simbol input (berbentuk jajar genjang) diisi dengan kata “Masukkan jari-jari”.Simbol input (berbentuk jajar genjang) diisi dengan kata “Masukkan tinggi”.Simbol proses (berbentuk persegi panjang) diisi dengan kata “Volume tabung = 3,14 * jari-jari * jari-jari * tinggi”.Simbol output (berbentuk jajar genjang) diisi dengan kata “Tampilkan volume”.Simbol terminal (berbentuk bangun sisi lengkung yang memanjang seperti perti panjang) diisi dengan kata “Selesai”.

Untuk lebih jelasnya dapat dilihat di lampiran.

27. contoh perhitungan flowchart sistem pencabangan tunggal dan ganda ? Pelajaran : Sistem Komputer



ALGORITMA PERCABANGAN 1 KONDISIAlgoritma percabangan dengan 1 kondisi memiliki format dasar seperti berikut :            IF <kondisi>            THEN <pernyataan>Pada format diatas , jika <kondisi> bernilai benar maka <pernyataan> dikerjakan, sedangkan jika bernilai salah, maka <pernyataan> tidak dikerjakan dan proses langsung keluar dari percabangan.Contoh Kasus : Nilai siswa akan gagal bila bernilai ≤ 60
ALGORITMA BAHASA NATURALMulaiMenentukan nilai siswaJika nilai siswa ≤ 60Siswa dinyatakan GagalSelesai
PSEUDOCODEVar :            nilai_siswa : integerPseudocode :Read ( nilai siswa )IF nilai siswa ≤ 60 THEN write ( Siswa dinyatakan gagal )
FLOWCHART



ALGORITMA PERCABANGAN 2 KONDISIAlgoritma percabangan dengan 2 kondisi memiliki format dasar seperti berikut :            IF <kondisi>            THEN <pernyataan 1>            ELSE <pernyataan 2>Pada format di atas, Jika <kondisi> bernilai benar maka pernyataan 1 dikerjakan. Sedangkan jika tidak (<kondisi>bernilai salah), maka pernyataan yang dikerjakan adalah pernyataan 2. Berbeda dengan percabangan 1 kondisi, pada percabangan dua kondisi ada dua pernyataan untuk kedua kondisi,yaitu untuk <kondisi> yang bernilai benar dan <kondisi> yang bernilai salah.Contoh Kasus : Penggolongan nilai
ALGORITMA BAHASA NATURALMulaiMenentukan nilai siswaJika nilai siswa ( 0 – 100 ) > 75 Tergolong BaikJika nilai siswa ( 0 – 100 ) ≤ 75 Tergolong CukupTampilkan hasilSelesai
PSEUDOCODEVar :            nilai_siswa : integerPseudocoderead (nilai siswa )IF nilai siswa > 75 THENwrite ( tergolong baik )ELSEwrite ( tergolong cukup )
FLOWCHART

36. Tuliskan dan gambarkan dalam tabel sistem Flowchart dan program Flowchart beserta Keterangannya.​


Jawaban :

Fungsi flowchart adalah digunakan untuk menganalisa, mendesain, mendokumentasikan dan memanajemen sebuah proses atau program di berbagai bidang. Secara khusus, flowchart berfungsi untuk membantu menggambarkan situasi apa yang sedang terjadi dan yang akan terjadi dari sebuah simbol dan tanda penghubungnya. Selain itu, flowchart ini mampu memperjelas sebuah alur dari suatu sistem baik itu kekurangan atau kelebihan dari berbagai proses di dalam tahapan suatu sistem.

Penjelasan:

Tujuan Flowchart

Berikut ini terdapat beberapa tujuan flowchart, diantaranya adalah:

Menggambarkan urutan atau tahapan dari penyelesaian masalah

Menggambarkan permasalahan secara sederhana, terurai, rapi dan jelas

Flowchart membantu analis dan programmer untuk memecahkan masalah ke dalam bagian-bagian yang lebih kecil dan menolong dalam menganalisis alternatif-alternatif lain dalam pengoperasian. Flowchart mempermudah penyelesaian suatu masalah khususnya masalah yang perlu dipelajari dan dievaluasi lebih lanjut.

Siklus dalam flowchart adalah siklus input-proses-output atau sering disebut siklus IPO. Siklus IPO dapat kita analogikan sebagai komponen hardware komputer. Komponen input seperti keyboard, mouse, scanner, dan sebagainya dapat kita andaikan sebagai input (penerimaan suatu perintah atau data yang akan diproses).

Komponen proses seperti CPU akan mengolah/memproses data yang telah masuk dalam komponen input agar menghasilkan suatu hasil. Dan akhirnya hasil pemrosesan data atau perintah yang dilakukan oleh CPU ditampilkan pada monitor, speaker atau printer.

Simbol-simbol yang di pakai dalam flowchart dibagi menjadi 3 kelompok :

1. Flow Direction Symbols

2. Processing Symbols

3. Simbol Input atau Output

40. berikan contoh flowchart,pseudocode beserta codingnya?


Problem: Buatlah program untuk memeriksa pemasukan lebih besar/lebih kecil dari pengeluaran!
Algoritma:
-Masukkan pemasukan
-Masukkan pengeluaran
-Jika pemasukan > pengeluaran maka pemasukan lebih besar dari pengeluaran
-Jika pemasukan < pengeluaran maka pemasukan lebih kecil dari pengeluaran
-tampilkan

Pseudocode:
-Input m
-Input k
-If m > k maka pemasukan lebih besar dari pengeluaran
-If m < k maka pemasukan lebih kecil dari pengeluaran
-Print
