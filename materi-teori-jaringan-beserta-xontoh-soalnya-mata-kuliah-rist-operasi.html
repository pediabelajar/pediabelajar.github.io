Jawab:

1 atau 12

Penjelasan dengan langkah-langkah:

Diketahui FPB(a, b) =1

Asumsikan FPB (2a+3b, 4a-3b) = d sehingga d|(2a+3b) dan d|(4a-3b)

Berdasar teorema keterbagian

d|4(2a+3b) - 2(4a-3b) = 12b

d|2(4a-3b) + 2(4a+3b) = 12a

Maka, FPB(12a, 12b)==> FPB 12(a,b)  ==> d|12 dan d|1

Jadi, dapat disimpulkan FPB  (2a+3b, 4a-3b) = 1 atau 12

Jawab:

1.  aljabar linear = 600 mahasiswa

2.  Bahasa pemograman =  300 mahasiswa

3.  sistem operasi I =  200 mahasiswa

Penjelasan dengan langkah-langkah:

jika ; Aljabar Linear = a

       bahasa pemograman = b

dan sistem operasi I = c

jika nilai a, b dan c selalu sama, maka dalam skala 1% didapat :

10a + 20b = 120 mahasiswa

6a + 15c = 60 mahasiswa

30b + 20c = 145 mahasiswa

⇔ {(10a + 20b = 120) x 3}  -  {(30b + 20c = 145) x 2}

⇔ (30a + 60b = 360)  -  (40c + 60b = 290)  =  30a - 40c = 70

⇒⇒ {(6a + 15c = 60) x 5}  -  (30a - 40c = 70)

⇒⇒ (30a + 75c = 300)   -   (30a - 40c = 70)   ⇔  115c = 230

⇒ 115c = 230  ⇒ c = 230/115 = 2

⇒⇒⇒ 30b + 20c = 145  ⇒ 30b + (20x2) = 145  ⇒  30b + 40 = 145  ⇒ 30b = 145 - 40  

⇒ 30b = 105  ⇒ b = 105/30  ⇒  b = 3

⇒ 10a + 20b = 120

⇒ 10a + (20 x 3) = 120

⇒ 10a + 60 = 120

⇒10a = 120 - 60

⇒10a = 60

⇒ a = 60/10 = 6

didapat nilai a = 6, b = 3 dan c = 2

maka jumlah masing-masing pada mata kuliah adalah:

1.  aljabar linear = a x 100 = 6 x 100 = 600 mahasiswa

2.  Bahasa pemograman = b x 100 = 3 x 100 = 300 mahasiswa

3.  sistem operasi I = c x 100 = 2 x 100 = 200 mahasiswa

__Pemuaian Panjang
Pemuaian panjang adalah pertambahan panjang benda akibat pengaruh suhu (1 dimensi). Coba amati kabel listrik yang terlihat lebih kendor di siang hari jika dibanding pada pagi hari, itulah contoh dari muai pemuaian panjang. Besarnya pemuaian zar tergantung pada konstanta muai panjang zat dan nilai konstanta tersebut akan berbeda-beda untuk tiap zatnya. Alat yang digunakan untuk menyelidiki pemuaian panjang berbagai jenis zat padat adalah musschenbroek. Pemuaian panjang suatu benda dipengaruhi oleh panjang mula-mula benda, besar kenaikan suhu, dan tergantung dari jenis benda.


Rumus pemuaian panjang
Δx= Lo. α. ΔT

ΔX =besarnya pemuaian panjang
Lo = panjang mula-mula
α = konstanta pemuaian
ΔT = selisih suhu
