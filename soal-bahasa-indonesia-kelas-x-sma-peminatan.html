3²ˣ ⁺ ¹ + 8 . 3ˣ - 3 > 0, HP = ?

_______________________________________

JAWABAN:

HP = { x | x > -1 }Opsi (B) x > -1

PENJELASAN:

⇔ Eksponen adalah bentuk bilangan berpangkat atas suatu bilangan basis yang berarti bilangan basis dikalikan dengan bilangan basis sebanyak pangkatnya.

⇔ Sifat-sifat eksponen yang digunakan:

aᵇ ⁺ ⁿ = aᵇ . aⁿ(aᵇ)ⁿ = aᵇ * ⁿaᵇⁿ = (aⁿ)ᵇ

PEMBAHASAN SOAL:

⇒ 3²ˣ ⁺ ¹ + 8 . 3ˣ - 3 > 0

Selesaikan menggunakan sifat eksponen:

3²ˣ ⁺ ¹ + 8 . 3ˣ - 3 > 03²ˣ . 3 + 8 . 3ˣ - 3 > 0(3ˣ)² . 3 + 8 . 3ˣ - 3 > 0

Substitusikan n = 3ˣ:

n² . 3 + 8n - 3 > 03n² +  9n - n - 3 > 03n(n + 3) - (n + 3) > 0(n + 3)(3n - 1) > 0n + 3 = 0 V 3n - 1 = 0n + 3 - 3 = 0 - 3 V 3n - 1 + 1 = 0 + 1n = -3 V 3n : 3 = 1 : 3n = -3 V n = 1/3

Pisahkan menjadi a > 0, b > 0 dan a < 0, b < 0:

n > -3n > 1/3HP = { n > 1/3 }n < -3n < 1/3HP = { n < -3 }HP = { n > 1/3 V n < -3 }

Masukkan nilai n = 3ˣ:

n = { n > 1/3 V n < 3 }3ˣ = { n > 1/3 V n < 3 }3ˣ < -33ˣ > 1/3

Selesaikan pertidaksamaan pertama:

3ˣ < -3Fungsi pemangkatan selalu positif, jika tidak maka penyelesaian bukan bilangan real.x ∉ R

Selesaikan pertidaksamaan kedua:

3ˣ > 1/33ˣ > 3⁻¹x > -1

Diperoleh:

x > -1, makaHP = { x | x > -1 }

26. Himpunan penyelesaian dari :sin² x - sin x = 0, 0 ≤ x ≤ 2π​​Matematika PeminatanSMA kelas 11


Himpunan penyelesaian :

sin x = 0 > x = 0, π, 2π

sin x = 1 > x = π/2

PEMBAHASAN

Menggunakan rumus persamaan trigonometri untuk fungsi sinus.

Trigonometri adalah materi matematika yang berhubungan dengan perbandingan antara sudut dengan sisi pada segitiga.

Persamaan trigonometri adalah persamaan yang memuat fungsi trigonometri tetapi dengan sudut yang belum diketahui.

Diketahui :

0 ≤ x ≤ 2π

Ditanya :

sin² x - sin x = ...?

Dijawab :

sin² x - sin x = 0

sin x (sin x - 1) = 0

sin x = 0 atau sin x = 1

sin x = 0 > x = 0, π, 2π

sin x = 1 > x = π/2

35. Apa perbedaan pelajaran bahasa inggris peminatan dan bahasa inggris wajib pada sma


Jawaban: aku bantu jawab ya kak!

Penjelasan: Perbedaan antara pelajaran Bahasa Inggris peminatan dan Bahasa Inggris wajib di SMA terletak pada fokus, konten, dan tingkat kompleksitas materi yang diajarkan. Berikut adalah penjelasan singkat mengenai perbedaan keduanya:

1. Bahasa Inggris Wajib: Pelajaran Bahasa Inggris wajib merupakan bagian dari kurikulum yang harus dipelajari oleh semua siswa di SMA. Pelajaran ini memberikan dasar-dasar Bahasa Inggris yang meliputi keterampilan membaca, menulis, mendengarkan, dan berbicara. Materi yang diajarkan cenderung lebih umum dan mencakup topik-topik seperti tata bahasa dasar, kosakata umum, percakapan sehari-hari, membaca teks sederhana, dan menulis esai singkat. Tujuan dari pelajaran ini adalah untuk memberikan pemahaman dasar Bahasa Inggris kepada siswa agar mereka dapat berkomunikasi secara efektif dalam situasi sehari-hari.

2. Bahasa Inggris Peminatan: Pelajaran Bahasa Inggris peminatan biasanya ditujukan untuk siswa yang memiliki minat dan kemampuan lebih dalam Bahasa Inggris. Sifat-sifat eksponen

[tex]{a}^{b} \times {a}^{c} = {a}^{b + c} \\ \\ \frac{ {a}^{b} }{ {a}^{c} } = {a}^{b - c} \\ \\ {a}^{c} \times {b}^{c} = {(a \times b)}^{c} \\ \\ {a}^{ {b}^{c} } = {a}^{( {b}^{c} )} \\ \\ {( {a}^{b}) }^{c} = {a}^{b \times c} \\ \\ [/tex]

[tex] {2}^{3x} = 64 \\ \\ [/tex]

[tex]\text{Nilai dari } \: \: 16 - {x}^{2} \\ \\ [/tex]

Jawab :

[tex] {2}^{3x} = 64 \\ \\ {2}^{3x} = {2}^{6} \\ \\ 3x = 6 \\ \\ x = \frac{6}{3} \\ \\ \boxed{x = 2} \\ \\ [/tex]

[tex] \: \: \: \: \: 16 - {x}^{2} \\ \\ = 16 - {(2)}^{2} \\ \\ = 16 - 4 \\ \\ = 12 \\ \\ [/tex]
