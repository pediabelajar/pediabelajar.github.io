Penjelasan dengan langkah-langkah:

no. 6

(x² - x + 3)/(x - 2) ≥ 0

karena x² - x + 3 merupakan definit positif (D < 0, a > 0). Maka, bisa dicoret saja menjadi :

1/(x - 2) ≥ 0

• Pembuat nol :

x = 2

• Syarat : nilai x ≠ 2

• Garis bilangannya :

––– (2) +++

Karena bertanda "≥", berarti arahnya ke positif.

Jadi, HP = {x | x > 2}

no. 7

(x² - 2x - 3)/(x² + 2x - 8) ≥ 0

(x + 1)(x - 3)/(x + 4)(x - 2) ≥ 0

• Pembuat nol :

x = -4, x = -1, x = 2, dan x = 3

• syarat : nilai x ≠ -4 dan x ≠ 2

• Garis bilangannya :

+++ (-4) ––– (-1) +++ (2) ––– (3) +++

karena bertanda "≥", berarti arahnya ke positif.

Jadi, HP = {x | x < -4 atau -1 ≤ x < 2 atau x ≥ 3}

no. 8

(x² - 3x - 4)/(x² - x - 2) < 0

(x + 1)(x - 4)/(x + 1)(x - 2) < 0

(x - 4)/(x - 2) < 0

• Pembuat nol :

x = 2 atau x = 4

• Garis bilangannya :

+++ (2) ––– (4) +++

karena bertanda "<", berarti arahnya ke negatif.

Jadi, HP = {x | 2 < x < 4}
