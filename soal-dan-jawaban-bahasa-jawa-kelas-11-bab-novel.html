7. soal kelas 11 bab turunan soal no 15


misal
sisi alas balok = s
tinggi balok = t

karena alas balok berbentuk persegi,
maka
luas balok = 96cm²
2×s² + 4st = 96
s² + 2st = 48
t = ( 48 - s² )/2s

volume balok = s² × t
V = s² * ( 48 - s² )/2s
V = 24s - ½s³

agar V maks, maka V ' = 0
24 - 3/2s² = 0
s² = 16
s = 4 cm

10. soal kelas 11 bab turunan soal no 18


keuntungan = penjualan total - biaya total
(penjualan total didapat dengan mengalikan nilai
penjualan per unit dgn x, yaitu banyaknya unit yang terjual)
K = 40x - (4x² - 8x + 24)x
K = -4x³ + 8x² - 24x + 40x
K = -4x³ + 8x² + 16x


agar keuntungan maksimum
K ' = 0
-12x² + 16x + 16 = 0
3x² - 4x - 4 = 0
(3x + 2)(x - 2) = 0
x = -2/3 atau x = 2

maka keuntungan maksimum tercapai ketika x = 2
yaitu
K = -4(2)³ + 8(2)² + 16(2)
= -32 + 32 + 32
= 32 (dalam ribuan)

jadi keuntungan maksimum = Rp32.000 ✔️

2ab + 3ab +7ab² - 9ab + 6c​

= (2ab + 3ab + (-9ab)) + 7ab² + 6c

= (5ab - 9 ab) + 7ab² + 6c
