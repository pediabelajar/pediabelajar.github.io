11. sebuah segitiga siku siku , jumlah panjang sisi siju sikunya adalah 49 cm. adapun panjang sisi miringnya 35 cm. panjang sisi siku siku segitiga tersebut adalah..jawab dengan cara ya


Pembahasan:
Sudah di ketahui
a + b + c = 49
diberikan c = 35 → a + b + 35 = 49
⇔a + b = 14
⇔ b = 14 - a ....(1)
Misalkan a,b adalah panjang sisi-sisi siku-siku , dan
c adalah panjang sisi miring segitiga tersebut,
Menurut dalil pythagoras berlaku:
a² + b² = c²
untuk c = 35 → a² + b² = 35²
⇔a² + b² = 1225 ......(2)
Sekarang, kita selesaikan sistim persaman (1) dan (2) :
b = 14 - a → a² + b² = 1225
a² + (14-a)² = 35²
a² + 196 - 28a + a² = 1225
2a² - 28a + 196 - 1225 = 0
2a² -28a - 1029 = 0
Kita gunakan rumus diskriminan D = b² - 4ac
pada persamaan ax² + bx + c = 0
untuk uji kebenaran bilangan nyata
(jika akar akar tidak nyata, maka tidak
terdapat penyelesaian) pada relasi soal
yang anda 'buat' ini:
Kita dapatkan:
D = (-28)² -4(2)(-1029)
D = 784 + 8234
D = 9016 > 0 ( akar-akar nyata)
sehingga
b = (-b ± √D)/(2a)
b = (2 ± √9016)/4
b = (2 ± 94,95)/4
b = -21,7375 ( tidak mungkin)
atau
b = 24.2375 ( tidak mungkin juga karena a + b = 14 )
Jadi, soal yang anda 'buat' tersebut tidak
ada solusi.
Jawab
Soal yang anda buat ini tidak ada penyelesaian.

Catatan:
1. Dalam matematika, jenis penyelesaian sistim persamaan terdiri dari:
a. Solusi tunggal
b. Solusi ganda
c. Tidak ada solusi


2. Soal di atas bisa di jawab dengan singkat gunakan teori bilangan,
kita tahu bahwa untuk segitiga siku-siku berlaku a² + b² ≤ (a + b)²,
untuk soal anda, a² + b² = 1225 dan
(a + b)² = 14² = 196 sehingga 1225 > 196 (tidak memenuhi a² + b² ≤ (a + b)² ), jadi soal yang kamu buat ini mustahil ( bertentangan dengan teorema).

27. contoh soal segitiga siku siku dalam kehidupan sehari-hari​


Jawaban dan Penjelasan dengan langkah-langkah:

Seorang petani memiliki sawah dengan panjang 8 meter dan lebar 6 meter. Ia ingin memasang pagar keliling lahan dengan menggunakan kawat yang membentuk segitiga siku-siku. Berapa panjang kawat minimal yang dibutuhkan?

Jawaban:

Untuk menyelesaikannya diperlukan rumus pytagoras  a² + b² = c², di mana c merupakan sisi miring atau hipotenusa, dan a serta b merupakan sisi-sisi lainnya.

Karena c adalah sisi miring dan a dan b adalah sisi lainnya, maka sebagai berikut:

a = 8 m

b = 6 m

c = ?

c² = a² + b²

c² = 8² + 6²

c² = 64 + 36

c² = 100

c = √100

c = 10 meter

Jadi, panjang kawat minimal yang dibutuhkan adalah 10 meter.
