5. ajari aku iya tentang matriks


Matriks dalam matematika merupakan kumpulan bilangan, simbol atau ekspresi berbentuk persegi panjang yang disusun menurut baris dan kolom. Bilangan-bilangan yang terdapat pada suatu matriks disebut dengan elemen atau disebut juga anggota dari suatu matriks. Matriks banyak dimanfaatkan untuk menyelesaikan berbagai permasalahan matematika misalnya dalam menemukan solusi masalah persamaan linear, transformasi linear yakni bentuk umum dari fungsi linear contohnya rotasi dalam 3 dimensi. Matriks juga seperti variabel biasa, sehingga matrikspun dapat dimanipulasi misalnya dikalikan, dijumlah, dikurangkan, serta didekomposisikan. Menggunakan representasi matriks, perhitungan dapat dilakukan dengan lebih terstruktur.

Pengertian dari  transpose matriks yaitu suatu  matriks yang dilakukan pertukaran antara dimensi kolom dan barisnya.

Definisi lain dari transpose matriks adalah sebuah matriks yang didapatkan dengan cara memindahkan elemen - elemen pada kolom menjadi elemen baris dan sebaliknya. 

Misalkan : Diketahui sebuah matriks A seperti dibawah ini :

A =

abc


Maka tranpose matriksnya adalah :

AT =

abc



Sifat - Sifat Matriks Transpose

Transpose matriks memiliki beberapa sifat yang menjadi dasar di dalam operasi perhitungan matriks, yaitu :

(A + B)T = AT + BT

(AT)T = A

λ(AT) = (λAT), bila λ suatu scalar

(AB)T = BT AT

1. Diketahui matriks-matriks berikut:

[tex] R = \begin{pmatrix} 5&2\\4&-3\end{pmatrix} \:\:\:\:\text{ dan }\:\:\:\: S = \begin{pmatrix} 5&\frac{1}{2}x\\ 4y&-3 \end{pmatrix} [/tex]

Jika [tex] R = S^T [/tex] , tentukan nilai [tex] x+y [/tex]

Jawab:

[tex] \begin{align} R &= S^T \\ \begin{pmatrix} 5&2\\4&-3\end{pmatrix} &= \begin{pmatrix} 5&\frac{1}{2}x\\ 4y&-3 \end{pmatrix}^T \\ \begin{pmatrix} 5&2\\4&-3\end{pmatrix} &= \begin{pmatrix} 5&4y\\ \frac{1}{2}x&-3 \end{pmatrix} \end{align} [/tex]

Diperoleh dari kesamaan nilai matriks di atas:

[tex] \begin{align} 2&=4y \\ y &= \dfrac{2}{4} = \dfrac{1}{2} \end{align} [/tex]

kemudian:

[tex] \begin{align} 4 &= \dfrac{1}{2}x \\ x &= 4\cdot 2 = 8 \end{align} [/tex]

Sehingga nilai [tex] x+y = 8 + \frac{1}{2} = 8 \frac{1}{2} = \frac{17}{2} [/tex]

1.  SortingDalam penyelesaian suatu masalah pasti terdapat banyak cara atau solusi-solusi yang dapat dilakukan, seperti halnya pembuatan program memiliki banyak tehnik atau algoritma yang dapat di gunakan salah satunya untuk kebutuhan SORTING atau PENGURUTAN kumpulan data-data. terdapat 4 algoritma atau tehnik dalam melakukan sorting.Straight Selection Sort. teknik sorting ini dibuat dengan cara melakukan pengecek’an 1 persatu, bila kita akan mengurutkan secara ascending maka kita lakukan pengecek’an nilai tempat yang pertama (index pertama pada array) bila lebih kecil daripada index berikutnya (index 1 dengan index 2, index 1 dengan index 3, ….. index 1 dengan index terakhi) maka kita lakukan pertukaran nilai pada array index tersebut. proses ini dilakukan terus menerus sampai pada pengecekan index terakhir – 1 dengan nidex terakhir.Selection Sort.Teknik sorting ini dibuat dengan cara melakukan pengecek’an 1 persatu, bila kita akan mengurutkan secara ascending maka kita lakukan pengecek’an nilai tempat yang pertama (index pertama pada array)kita bandingkan dengan semua nilai yang ada kita cari nilai minimalnya. lalu simpan index/ letak nilai minimum itu di temukan, setelah pengecekan selesai tukar index awal pengecekan dengan nilai minimum yang telah di simpan tadi. Proses ini dilakukan terus menerus sampai pada pengecekan index terakhir min 1 dengan index terakhir. beda dengan streith selection sort adalah dengan teknik ini melakukan pertukaran nilai lebih sedikit, hanya jumlah data – 1 pertukaran. jadi waktu untuk melakukan proses sorting lebih cepat.Bubble Sort. Teknik ini dilakukan degan pola membawa nilai terbesar menjadi nilai index terakhir array. jadi sistem ini melakukan pengecekan nilai 1 dengan 2, lalu 2 dengan 3 samapai dengan data terakhir, bila nilai index yang lebih kecil lebih besar maka akan dilakukan pertukaran. proses ini dilakuan hingga jumlah data – 1.Modified Bubble Sort. Teknik ini dilakukan degan pola membawa nilai terbesar menjadi nilai index terakhir array. Jadi sistem ini melakukan pengecekan nilai 1 dengan 2, lalu 2 dengan 3 samapai dengan data terakhir, bila nilai index yang lebih kecil lebih besar maka akan dilakukan pertukaran. proses ini dilakuan hingga jumlah data dikurangi 1 atau sampai program tidak melakukan pertukaran. jadi waktu untuk melakukan proses sorting lebih cepat.

#define _CRT_SECURE_NO_WARNINGS

#include<stdio.h>

void printArray(int cols,int rows,int matrix1[3][2],int matrix2[3][2])

{

printf("Matrix 1:\n");

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  printf("%d ",matrix1[i][j]);

 }

 printf("\n");

}

printf("\nMatrix 2:\n");

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  printf("%d ",matrix2[i][j]);

 }

 printf("\n");

}

}

void swapArray(int cols,int rows,int matrix1[3][2],int matrix2[3][2])

{

int temp;

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  temp = matrix1[i][j];

  matrix1[i][j] = matrix2[i][j];

  matrix2[i][j] = temp;

 }

 

}

}

int main()

{

int cols = 2;

int rows = 3;

int matrix1[3][2] =  

{

 {3,1},

 {4,2},

 {5,3}

};

int matrix2[3][2] =  

{

 {9,4},

 {6,8},

 {7,1}

};

printArray(cols,rows,matrix1,matrix2);

swapArray(cols,rows,matrix1,matrix2);

printf("\nAfter Swap\n\n");

printArray(cols,rows,matrix1,matrix2);

getchar();

return 0;

}

Penjelasan:

Program ini pertama menginisialisasi jumlah baris dan kolom. Kemudian matrix1 dan matrix2 diinisialisasi dengan nilai yang telah ditentukan. Untuk mengetahui perbedaan antara sebelum dan sesudah maka dibuatlah method printArray untuk mencetak matrix1 dan matrix2. Method printArray digunakan 2 kali untuk melihat nilai sebelum dan sesudah. Terakhir ada method untuk swap nilai matrix 1 dan matrix 2 dengan menggunakan bantuan variable temp untuk menyimpan nilai matrix 1 sebelum dioverwrite nilai matrix2 dan digunakan untuk mengoverwrite nilai matrix2.
