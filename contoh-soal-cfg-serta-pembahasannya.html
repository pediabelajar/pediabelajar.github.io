Segitiga sembarang ABC

E tengah BC

D tengah AC

G tengah EC

F tengah FC

Luas segitiga ABC

L.ABC = 1/2 × AC × BC × sin C

Luas segigiga CFG

L.CFG

= 1/2 × GC × FC × sin C

= 1/2 × 1/4 BC × 1/4 AC × sin C

= 1/16 × 1/2 × AC × BC × sin C

= 1/16 L.ABC

Luas segitiga CFG = 1/16 luas segitiga ABC

sudut BCF bertolak belakang dengan sudut ACD,

sehingga sudut BCF = sudut ACD = (3x)°

sudut BCF dan sudut CFG adalah sudut dalam berseberangan,

sehingga sudut BCF = sudut CFG = (x+40)°

maka,

sudut ACD = sudut BCF = sudut CFG

jadi,

(3x)° = (x+40)°

3x = x + 40

3x - x = 40

2x = 40

x = 40/2

x = 20

jadi,

sudut BCF = sudut ACD = 3x

= 3(20)

= 60°

sudut CFG = x + 40

= 20 + 40

= 60°

24. 10. Kubus ABCDEFGH mempunyai panjang 6 cm. Jarak antara bidang BDE dan CFG adalah A. 2√2 B. 2√3 C. 3√3 D. 6√2 E. 6√2​


Jawaban:

C. 3√3.

Penjelasan dengan langkah-langkah:

Untuk menjawab pertanyaan tersebut, mari kita analisis informasi yang kita miliki.

Diberikan bahwa kubus ABCDEFGH memiliki panjang 6 cm.

Dalam kubus, diagonal antara bidang yang saling sejajar memiliki panjang yang sama dengan panjang sisi kubus. Jadi, diagonal antara bidang BDE dan CFG adalah juga 6 cm.

Jarak antara dua bidang sejajar adalah jarak titik-titik terjauh pada kedua bidang tersebut. Dalam kasus ini, kita mencari jarak antara bidang BDE dan CFG.

Kedua bidang tersebut berada pada sisi kubus yang berseberangan. Jadi, jarak antara kedua bidang tersebut adalah jarak antara dua diagonal kubus yang saling berseberangan.

Dalam kubus ABCDEFGH, diagonal kubus yang saling berseberangan akan melalui pusat kubus. Dalam hal ini, jarak antara bidang BDE dan CFG adalah diagonal ruang kubus.

Diagonal ruang kubus panjangnya dapat dihitung menggunakan teorema Pythagoras dengan menggunakan panjang rusuk kubus sebagai sisi.

Panjang diagonal ruang kubus = √(sisi^2 + sisi^2 + sisi^2) = √(6^2 + 6^2 + 6^2) = √(36 + 36 + 36) = √108 = 6√3 cm

Jadi, jarak antara bidang BDE dan CFG adalah 6√3 cm.

Opsi jawaban yang benar adalah C. 3√3.


25. CFG S -> cAd A -> ab | e String : ced 1. Tentukan LR 2. Tentukan SLR


Untuk mengidentifikasi LR (Left-to-Right, Rightmost derivation) dan SLR (Simple LR) dari tata bahasa yang diberikan, kita perlu membangun tabel parsing untuk tata bahasa tersebut.

Grammar CFG (Context-Free Grammar) yang diberikan:

S -> cAd

A -> ab | e

String yang diberikan: ced

1. LR (Left-to-Right, Rightmost derivation):

Berikut adalah derivasi LR dari string "ced" dengan menggunakan produksi-produksi yang ada dalam grammar CFG tersebut:

S -> cAd (Mulai dengan produksi S)

S -> ceAd (Ganti terminal "c" dengan "ce")

S -> ced (Ganti terminal "A" dengan "e")

Jadi, derivasi LR dari string "ced" adalah: S -> ced.

2. SLR (Simple LR):

Untuk menentukan SLR (Simple LR), kita perlu membangun tabel parsing SLR dari grammar CFG yang diberikan.

Berikut adalah langkah-langkah untuk membangun tabel parsing SLR:

Langkah 1: Membuat himpunan berisi produksi-produksi yang ada dalam grammar CFG:

P0: S' -> S

P1: S -> cAd

P2: A -> ab

P3: A -> e

Langkah 2: Membuat himpunan berisi semua terminal yang muncul dalam grammar CFG:

T = {c, a, b, e, d}

Langkah 3: Membuat himpunan berisi semua non-terminal yang muncul dalam grammar CFG:

N = {S', S, A}

Langkah 4: Membuat himpunan berisi semua state dalam tabel parsing SLR:

I0: S' -> .S (Closure)

      S -> .cAd (from P0)

      A -> .ab (from P2)

      A -> .e (from P3)

I1: S' -> S. (from I0)

I2: S -> c.Ad (from P1)

      A -> .ab (from P2)

      A -> .e (from P3)

I3: A -> a.b (from P2)

I4: A -> ab. (from P2)

I5: A -> e. (from P3)

Langkah 5: Membuat tabel parsing SLR:

Tabel parsing SLR akan memiliki baris untuk setiap state dalam himpunan state, dan kolom untuk setiap terminal dan non-terminal dalam himpunan terminal (T) dan non-terminal (N). Isi tabel tersebut akan berisi informasi berdasarkan perpindahan (shift) dan reduplikasi (reduce) yang dilakukan dalam parsing SLR.

Berikut adalah tabel parsing SLR yang telah dibangun untuk grammar CFG yang diberikan:

```

 |  c  |  a  |  b  |  e  |  d  |  $  |

--------------------------------------

I0|  S2 |     |     |     |     |     |

I1|     |     |     |     |     |  Acc|

I2|     |     |     |     |  S3 |     |

I3|     |  S4 |     |     |     |     |

I4|     |  R2 |  S5 |     |     |     |

I5|     |  R3 |     |     |     |     |

```

Keterangan:

- Sx: Pindah ke state x.

- Rx: Reduksi dengan produksi ke-x

.

Dari tabel parsing SLR, kita dapat melakukan langkah-langkah parsing SLR untuk string "ced" dengan memanfaatkan tabel tersebut.

Jadi, untuk string "ced":

- Pindah dari I0 ke I2 dengan terminal "c".

- Pindah dari I2 ke I5 dengan terminal "e".

- Reduksi dari I5 dengan produksi P3 (A -> e).

Dengan langkah-langkah tersebut, string "ced" akan diterima oleh grammar CFG yang diberikan menggunakan parsing SLR.
