30. Saya seorang siswa hal yang saya sukai pemrograman ada dua bahasa yang saya sukai c++ sama html ... apa yang lebih penting


Jawaban :

Bahasa C++ merupakan bahasa pemrograman high-level programming language. Bahasa C++ merupakan bahasa pengembangan dari bahasa pendahulunya yaitu bahasa C. Penggunaan bahasa C++ bisa digunakan untuk mengembangkan aplikasi.

HTML (Hyper Text MarkUp Language) merupakan bahasa markup. HTML bukan merupakan bahasa pemrograman karena HTML hanya terdiri dari tag-tag HTML.

Penjelasan lebih lanjut:

Bahasa C++ diciptakan oleh Bjame Stroustrup pada tahun 1980. Kelebihan dari Bahasa C++ adalah Bahasa C++ mendukung object oriented programming (OOP) / Pemrograman Berorientasi Objek.

Bahasa pemrograman merupakan suatu media/perantara untuk berkomunikasi dengan mesin/assembler. Bahasa pemrograman terbagi menjadi 3 (tiga) tingkatan bahasa, yaitu : low-level programming language, middle-level programming language dan high-level programming language.

Contoh low-level programming language adalah bahasa assembly yang merupakan bahasa mesin pada komputer.

Contoh middle-level programming language adalah bahasa Cobol, Fortran, Pascal dan lain-lain.

Contoh high-level programming language adalah bahasa Java, C++, C#, dan lain-lain.

Compiler merupakan suatu platform untuk menerjemahkan bahasa pemrograman menjadi bahasa mesin sehingga menjadi program yang dapat digunakan. Compiiler digunakan untuk membaca bahasa pemrograman dengan cara keseluruhan baris dan diterjemahkan menjadi program.
