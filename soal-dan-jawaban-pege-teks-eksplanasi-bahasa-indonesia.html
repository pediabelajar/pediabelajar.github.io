18. jelaskan fungsi group yang terdapat pada tab pege layout


Page layout, kalau diartikan bahasa indonesia adalah tata letak halaman, Jadi, sudah sangat jelas, Untuk fungsi dari page layout adalah bekenaan dengan tata letak halaman kerja di microsoft word.

Seperti yang kita ketahui bersama, Bahwasanya, Page layout terbagi menjadi 5 group yaitu themes, Page setup, Page background, Paragraph dan Arrange. dan masing-masing dari group  tersebut mempunyai icon-icon berbeda beda. dan, tentu saja, dari setiap icon dari group tersebut, mempunyai fungsi yang berbeda beda pula.

Mungkin Anda belum tau: Fungsi dari icon icon office button 
Gambar halaman tab layout microsoft word 2007
Nah, untuk lebih mengenal fungsi dari icon-icon pada page layout. penjelasanya sebagai berikut:
1. Fungsi icon pada group Themes.Gambar group themesThemes: digunakan untuk menganti tema halaman kerja.Color: digunakan untuk mengganti warna teks dan background.Font: digunakan untuk mengubah model teks.Effect: digunakan untuk menambahkan effek warna teks dan background.2. Fungsi icon pada group Page setup.Gambar group page setupMargins: digunakan untuk menentukan jarak tulisan dari samping kiri dan kanan halaman kerja.Orientation: digunakan untuk menentukan halaman kerja (kertas)horizontal atau vertikal.Size: digunakan untuk menentukan ukuran halaman kerja atau kertas.Columns: digunakan untuk menentukan jumlah kolom di dalam halaman kerja.Breaks: digunakan untuk menentukan jarak antar kolom halaman.Line number: digunakan untuk menambahkan nomor di setiap baris teks.Hyphenation: digunakan untuk memisahkan suku kata dengan tanda penghubung.
3. Fungsi icon pada group Page background.Gambar group page backgroundWatermark: digunakan untuk menambahkan tanda air di background halaman menggunakan teks atau gambar.Page color: digunakan untuk mengubah warna halaman kerja menggunakan warna.Page borders: digunakan untuk membuat garis di pinggir halaman kerja.
4. Fungsi icon pada group Paragraph
Gambar group paragraphIndent: digunakan untuk mengatur jarak teks kiri dan kanan.Spacing: digunakan untuk mengatur jarak antara atas dan bawah paragraph.
5. Fungsi icon pada group Arrange.gambar group arrangePosition: digunakan untuk menentukan posisi objek gambar.Bring to fornt: digunakan untuk menentukan posisi objek lain dengan yang lainya.Send to back: digunakan untuk mengembalikan objek lain dengan yang lainya( kebalikan dari Bring to fornt).Text wrapping: digunakan untuk meratakan teks supaya mengikuti objek.Align: digunakan untuk meluruskan posisi objek.Group: digunakan untuk menghubungkan antar objek satu dengan yang lainya supaya lebih mudah mengatur posisi objek.Rotate: digunakan untuk memutar posisi objek gambar.

6. Apa isine teks iklan ing dhuwur?

➡️ Medharake bab obat herbal kanggo masuk angin kang arane "Tolak Angin Komplit".

7. Apa tujuan sing diancas kanthi iklan kasebut?

➡️ Ancase nawari ngunjuk jamu Tolak Angin Komplit supaya awak dadi seger maneh.

8. Apa medhia sing cocok kanggo nggiyarake teks iklan ing dhuwur?

➡️ Medhia elektronik, kayata radhio.

9. Terangna apa bedane iklan kang migunakake medhia cithak lan medhia elektronik!

➡️ Medhia cithak awujud salebaran, pamflet, baliho, ariwarti, lan kalawarti. Dene medhia elektronik yaiku televisi, radhio, lan internet.

10. Kepriye Panemuku ngenani basa sing dienggo ing iklan kasebut?

➡️ Miturut panemuku basane informatif, persuasif, komunikatif, lan sugestif. Basa ing njero iklan wis bener, cekak aos, pantes, mentes lan nengsemake.

Penjelasan

Iklan utawa pariwara yaiku tulisan (bisa dikantheni gambar utawa swara) kang nawakake barang, jasa utawa ajak-ajak. Nawakake tegese ngandhakake barang kang arep didol, kaluwihane apa, lan cara nggunakake kepriye. Dene ajak-ajak karepe supaya wong sing maca utawa krungu gelem nganggo kanthi cara tuku barang kang ditawakake.

25. insert pege leyout home termaksud dalam program ms wordcepat ya sekarang di kumpul​


- Home

Home merupakan menu utama yang akan tampil pada saat pertama kali anda membuka word 2007. Menu home terdiri dari toolbar standar yaitu font, tool bar formatting dan paragraph, style dan editing serta clipboard.

- Insert

Tab ini digunakan untuk memasukkan instruksi-instruksi ke dalam dokumen. Menu insert terdiri dari : Pages, Tables, Illustrations, Links, Header & Footer, Text, Symbol.

- Page Layout

Tab page Layout digunakan untuk mengatur tata letak data, yang meliputi Page Setup, memberiwarna background pada lembar kerja, mengatur spasi antar baris, membuat kolom pada teks dan mengatur indentasi. Menu page layout terdiri dari : Themes, Page
