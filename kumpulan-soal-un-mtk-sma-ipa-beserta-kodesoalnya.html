28. kak mau bertanya, untuk soal test placement masuk SMA IPA/IPS itu soal seperti apa dan tentang apa ipa,mtk,bindo..? mohon bantuanya​


test placement itu adalah test untuk menentukan jurusan di SMA.

ada dua jurusan di tingkat SMA, yaitu

IPA dan IPS.

(beberapa sekolah ada yg memiliki jurusan bahasa)

test placement setiap sekolah bisa saja berbeda-beda.

ada yg berupa test biasa (mata pelajaran penting seperti matematika, IPA, IPS, bahasa Indonesia, dan bahasa Inggris)

nilai mata pelajaran yg paling tinggi dapat menentukan jurusan Yanga akan kamu dapatkan/pilih. misalnya nilai IPS dan bahasa Inggris kamu lebih besar daripada nilai yang lain, maka kamu kemungkinan akan diposisikan di jurusan IPS.

ada juga yang menggunakan test Psikotes untuk menentukan jurusan di SMA. hasil test ini didapatkan dari pertimbangan oleh psikolog yang merekomendasikan kamu dalam memilih jurusan.

32. soal dari buku detik-detik UN matematika SMA IPA kategori : trigonometri soal nomor 28


Kelas            : 11
Mapel           : Matematika
Kategori      : Trigonometri lanjut
Kata kunci  : trigonometri, aturan cosinus
Kode            : 11.2.3 (Kelas 11 Matematika Bab 3-Trigonometri lanjut)

Gunakan aturan cos untuk segitiga ABD:
BD²=AB²+AD²-2.AB.AD. cos A

Gunakan aturan cos untuk segitiga BCD:
BD²=BC²+CD²-2.BC.CD. cos C

Berdasarkan sifat segiempat tali busur, sudut A dan C berjumlah 180° (karena berhadapan)

BD²=BD²
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²-2.BC.CD. cos C
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²-2.BC.CD. cos (180°-A)
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²+2.BC.CD. cos A
AB²+AD²-BC²-CD² = 2.BC.CD. cos A+2.AB.AD. cos A
AB²+AD²-BC²-CD² = cos A (2.BC.CD+2.AB.AD)
cos A = (AB²+AD²-BC²-CD²) / (2.BC.CD+2.AB.AD)
cos A = (4²+6²-3²-3²) / (2.3.3+2.4.6)
cos A = (16+36-9-9) / (18+48)
cos A = 34/66
cos A= 17/33

Jadi, cos ∠BAD adalah 17/33

Jawaban : C
