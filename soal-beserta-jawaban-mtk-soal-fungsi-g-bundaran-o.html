26. Diketahui fungsi gx = 2 x + 3 dan fx = x per X kurang 2 untuk X tidak sama dengan 2 apabila G Bundaran F pangkat min 1 x merupakan invers dari G Bundaran FX rumus fungsi dari g Bundaran F pangkat min 1 x adalah


Invers dari suatu fungsi f(x) dinyatakan dengan f⁻¹(x).

(f o g) (x) = f(g(x))

Pembahasan

g(x) = 2x + 3

f(x) = [tex] \frac{x}{x - 2} [/tex], dengan x ≠ 2

(g o f) (x) = g(f(x))

              = 2(f(x)) + 3

              = 2([tex] \frac{x}{x - 2} [/tex]) + 3

              = [tex] \frac{2x}{x - 2} + 3 [/tex]

              = [tex] \frac{(2x) + 3 (x - 2) }{x - 2} [/tex]
