2. tolong di bantu yaa bahasa inggris tentang error analysis, pertanyaannya yg di stabilo'in berserta penjelasannya yaa ☺


1. d. places > harusnya place
karena Bali itu cuma satu tempat, singular
kalau places kan plural

2. b. are interesting > harusnya are interested
karena if you are interesting itu artinya kalau kamu menarik
kan gak nyambung
kalau are interested itu artinya kalau kamu tertarik

3. b. who have > harusnya who has
karena Uncle Tod itu singular, kalau subjek singular, PASTI pasangannya has

4. c. my > harusnya mine
karena dia sedang merujuk pada pensil miliknya, jadi dia pakai possessive pronoun
my: aku
mine: milikku

5. ini bener, tapi mungkin gurumu pikir kalau c. smoking itu salah dan harusnya to smoke

6. c. wherever > harusnya whenever
karena wherever itu artinya: di mana pun, kan gak nyambung sama kalimat
harusnya whenever, artinya kapanpun

7. a. was > harusnya were
if clause type 2, subjek PASTI diikuti dengan were, gak peduli singular atau plural

8. c. whose > harusnya which
whose itu digunakan buat orang, which untuk benda mati
motorcycle kan benda mati

9. a. should > harusnya would
should  = harus, gak nyambung sama kalimat
would = maukah
dan di mana2 itu pasangannya would you like to

9. contoh soal error analysis beserta pembahasannya


Jade : The first person to land on the moon was Sir Isaac Newton!

Rebecca: No! It was Soekarno!

Emily: Well.. In fact, it was Neil Armstrong who first landed on the moon. He landed with the Apollo mission team

Which words show disagreement?
A. Person + No!
B. Soekarno+Sir Isaac Newton
C. No! + In fact
D. The moon + landed

Jawabannya C, soalnya Rebecca menyatakan tidak setuju dengan Jade dan merasa bahwa dia salah dengan kata "No!", Sedangkan Emily merasa tidak setuju dengan keduanya dan menganalisa kesalahan (error) keduanya, dengan kata "In fact" (Sebetulnya/sebenernya)

Contoh : My mom cooks my breakfast jadi My breakfast is cooked by my mom
Klo sentencenya present continous, rumusnya to be + being + V3
Contoh : They are building house jadi Houses are being built

Contoh : Someone has found my dog jadi My dog has been found

Contoh :Everyone can make pizza jadi Pizza can be made

36. rumus kalimat aktif dan kalimat passive dalam bahasa inggris


Simple Present
Aktive :
(+) S + V1/V1 + s/es + Complement
(-) S + do/does + not + V1 + Complement
(?) Do/Does + S + V1 + Complement

Passive :
(+) O + am/is/are + V3 + Complement
(-) O + am/is/are + not + V3 + Complement
(?) Am/Is/Are + O + V3 + Complement

Present Continous
Aktive :
(+) S + am/is/are + V-ing + Complement
(-) S + am/is/are + not + V-ing + Complement
(?) Am/Is/Are + S + V-ing + Complement

Passive :
(+) O + am/is/are + being + V3 + Complement
(-) O + am/is/are + not + being + V3 + Complement
(?) Am/Is/Are + O + being + V3 + Complement

Present Perfect
Aktive :
(+) S + have/has + V3 + Complement
(-) S + have/has + not + V3 + Complement
(?) Have/Has + S + V3 + Complement

Passive :
(+) O + have/has + been + V3 + Complement
(-) O + have/has + not + been + V3 + Complement
(?) Have/Has + O + been + V3 + Complement


Present Perfect Continous

(+) S + have/has + been + V-ing + Complement
(-) S + have/has + not + been + V-ing + Complement
