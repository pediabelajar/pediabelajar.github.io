Pidato persuasif adalah sebuah pidato yang bersifat mengajak, membujuk, meyakinkan, dan melakukan hal yang diinginkan oleh orang yang berpidato tersebut.

PEMBAHASAN

Pidato secara umum bisa digambarkan suatu kegiatan berbicara didepan umum untuk menyampaikan argumen yang dilakukan oleh orator [orang yang berpidato] di depan banyak orang. Pidato ini biasanya menyampaikan tentang suatu hal untuk perlu diperbincangkan, seperti menyampaikan suatu argumen bagaimana cara mencegahnya banjir, dll.

Tujuan berpidato:

Memberikan suatu kesan kepada orang banyak.Memengaruhi seseorang agar mau mengikuti kemauan kita setelah berpidato [biasanya yang paling sering yaitu pidato persuasif]

Struktur teks pidato:

Salam pembuka, yang berisi salam untuk membuka/memulai suatu kegiatan berpidato. Seperti ucapan syukur kepada Tuhan Yang Maha Esa, Pemberian hormat kepada hadirin, dll. Isi, biasanya dalam isi pidato menyangkut hal hal yang dibahas.Penutup. Biasanya berisi akhir dari sebuah kegiatan pidato, misalkan kesimpulan dari apa yang dibahas, permohonan maaf, dan salam penutup.

» PIDATO PERSUASIF

Pidato persuasif merupakan sebuah pidato yang tujuannya untuk mengajak, membujuk, meyakinkan, juga mempengaruh agar para pendengar mau melakukan apa yang disampaikan oleh pembaca pidato tersebut.

Struktur teks pidato persuasif:

Pernyataan PosisiTahap argumenPenguatan Pernyataan posisi

Biasanya ada kata kata persuasif atau bersifat mengajak seperti Marilah, Ayolah, dll.Memakai kalimat yang bersifat membangun suasana atau lebih dikenal dengan kata emotif, supaya pendengar lebih terpakau. Menggunakan bahasa evaluatif.Menggunakan kata benda dan juga kata sifat.

13. materi lengkap tentang koleksi data dengan web scarping​


Jawaban:

Web scraping adalah proses mengambil data dari website secara otomatis menggunakan perangkat lunak atau script. Ini digunakan untuk mengumpulkan data dari berbagai sumber di internet dan digunakan dalam berbagai bidang, termasuk analitik bisnis, riset pasar, pemasaran, dan banyak lagi.

Untuk melakukan web scraping, seseorang harus memahami dasar-dasar HTML dan JavaScript. HTML (Hypertext Markup Language) digunakan untuk membuat struktur halaman web, sementara JavaScript digunakan untuk menambahkan interaksi dan dinamika pada halaman web.

Ada beberapa library yang digunakan dalam web scraping, seperti:

Beautiful Soup (Python)Scrapy (Python)Selenium (Python)Cheerio (JavaScript)Puppeteer (JavaScript)

Beautiful Soup adalah library Python yang digunakan untuk mengambil data dari HTML dan XML. Ini memungkinkan Anda untuk mencari elemen tertentu dalam halaman web dengan mudah.

Scrapy adalah framework Python yang digunakan untuk mengambil data dari website. Ini memungkinkan Anda untuk mengatur pengambilan data secara cepat dan efisien.

Selenium adalah library Python yang digunakan untuk mengontrol browser. Ini digunakan untuk mengambil data dari website yang menggunakan JavaScript untuk memuat konten.

Cheerio adalah library JavaScript yang digunakan untuk mengambil data dari HTML. Ini memungkinkan Anda untuk mencari elemen tertentu dalam halaman web dengan mudah.

Puppeteer adalah library JavaScript yang digunakan untuk mengontrol browser. Ini digunakan untuk mengambil data dari website yang menggunakan JavaScript untuk memuat konten.

Selain itu, saat melakukan web scraping, penting untuk diingat bahwa beberapa website mungkin memiliki aturan yang melarang penggunaan web scraping. Oleh karena itu, sebelum melakukan web scraping, pastikan untuk membaca syarat dan ketentuan dari website yang akan digunakan. Selain itu, pastikan untuk tidak mengambil data yang tidak diinginkan atau melakukan aktivitas ilegal.

21. materi tentang web php​


Jawaban:

Pengertian PHP, PHP Adalah bahasa scripting server-side, Bahasa pemrograman yang digunakan untuk mengembangkan situs web statis atau situs web dinamis atau aplikasi Web. PHP singkatan dari Hypertext Pre-processor, yang sebelumnya disebut Personal Home Pages.

Script sendiri merupakan sekumpulan instruksi pemrograman yang ditafsirkan pada saat runtime. Sedangkan Bahasa scripting adalah bahasa yang menafsirkan skrip saat runtime. Dan biasanya tertanam ke dalam lingkungan perangkat lunak lain.

Karena php merupakan scripting server-side maka jenis bahasa pemrograman ini nantinya script/program tersebut akan dijalankan/diproses oleh server. Berbeda dengan javascript yang client-side.

PHP adalah bahasa pemrograman umum yang berarti php dapat disematkan ke dalam kode HTML, atau dapat digunakan dalam kombinasi dengan berbagai sistem templat web, sistem manajemen konten web, dan kerangka kerja web.

Perbedaan Bahasa Pemrograman dan Bahasa Scripting

Bahasa Pemrograman VS Bahasa Scripting

Terdapat perbedaan antara bahasa pemrograman dan bahasa scripting, Apa saja perbedaan diantara keduanya baik dari segi kelebihan dan kekurangan:

Bahasa Pemrograman :

Bahasa pemrograman memiliki semua fitur yang diperlukan untuk membangun dan mengembangkan aplikasi yang lengkap.

Kode harus dikompilasi terlebih dahulu sebelum dapat dieksekusi.

Tidak perlu disematkan ke bahasa lain.

Bahasa Scripting

Bahasa scripting Paling banyak digunakan untuk tugas-tugas rutin

Kebanyakan kode ini biasanya dijalankan tanpa adanya kompilasi

Biasanya tertanam ke dalam lingkungan perangkat lunak lain.

Apa Fungsi PHP

Fungsi PHP adalah membuat atau mengembangkan situs web statis atau situs web dinamis atau aplikasi Web. Walaupun sebenarnya bukan hanya PHP bahasa pemrograman yang bisa digunakan untuk memuat website.

PHP digunakan karena untuk membuat website dinamis bisa digunakan untuk menyimpan data ke dalam database, membuat halaman yang dapat berubah-ubah sesuai dengan input user, memproses form, dll.

Dalam membuat file PHP dapat digabung menggunakan tag html, Dan ketika tanpa menggunakan tag html apa pun disebut file PHP Murni. Server menginterpretasikan kode PHP dan mengeluarkan hasilnya sebagai kode HTML ke browser web. Agar server mengidentifikasi kode PHP dari kode HTML, kita harus selalu menyertakan kode PHP dalam tag PHP.

Syntax PHP

Pengertian Syntax PHP adalah aturan penulisan agar mampu dimengerti dengan benar oleh compiler saat membaca bahasa pemrograman. Dalam penulisan PHP yang benar diawali dengan “<?php” dan diakhiri dengan “?>”. Dan di dalam File PHP juga dapat berisi tag seperti HTML dan skrip sisi klien seperti JavaScript.

Contoh Penulisan Syntax PHP yang Benar

<?php

echo ‘Hello World’ ;

?>

Dari kode program diatas akan keluar output Hello World. Berikut contoh lain penggunaan PHP yang disematkan kedalam HTML

PPTP menjadi pilihan populer jaringan virtual private network karena kemudahan dalam setting maupun penggunaan. Kamu tidak perlu melakukan instalasi VPN client agar bisa terhubung dengan jaringan server PPTP VPN. Koneksi Point to Point Tunneling Protocol bisa berjalan di berbagai sistem operasi seperti Microsoft Windows, Linux, MacOS X, iPhone OS, dan Android OS sudah memiliki built in protokol ini.

Kamu hanya membutuhkan nama server, vpn account meliputi nama pengguna dan kata sandi untuk dapat terhubung dengan server PPTP VPN. Teks laporan percobaan merupakan suatu teks yang berisikan paparan hasil percobaan yang telah dilakukan.

PEMBAHASAN

Teks laporan percobaan berguna untuk menginformasikan kebanyak orang mengenai hasil pengamatan, percobaan, serta laporan praktikum. Percobaan ini dilakukan untuk memahami suatu materi dengan cara melakukan suatu percobaan.

Teks laporan percobaan sering dikaitkan dengan teks deskripsi, karena keduannya sama-sama menginformasikan hasil pengamatan, analisa, ataupun percobaan. Perbedaan keduanya dapat diliat dari sifatnya, jika teks deskripsi menggambarkan suatu objek dengan hubungan panca indra. Saat membaca teks deskripsi, pembaca dapat menggunakan penglihatan, pendengaran, perabaan, penciuman, ataupun pengecapan dalam mendapatkan kesan pada objek yang dibahas. Hal tersebut merujuk kepada detail dari sifat maupun fisik dari objek. Artinya, teks deskripsi bersifat khusus dan unik.

Sedangkan teks laporan percobaan, memiliki sifat universal/umum. Teks laporan lebih mengedepankan pengelompokan ataupun klasifikasi dari suatu objek dalam percobaan.

Ciri-ciri

Teks laporan percobaan memiliki beberapa ciri-ciri yang membedakannya dengan teks lainnya, antara lain:

Tersusun atas fakta yang terjadi, bukan hanya opini/pendapat belaka,Tersusun berdasarkan percobaan yang dilakukan, Umumnya, bahasa yang digunakan baku serta bersifat ilmiah, Dikemukakan secara objektif, tidak berpihak pada siapapun.
